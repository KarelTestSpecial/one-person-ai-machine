# WORKFLOW: The Dual Gatekeepers

## 1. The Integrity Critic
- **Role**: Technical Gatekeeper.
- **Rules**: Zero broken links, Zero dead buttons, Proper SEO.
- **Fail Action**: Block build, log technical debt.

## 2. The Aesthetic Critic
- **Role**: Design Guardian.
- **Rules**: Glassmorphism mandatory, Premium Typography, Hover Anims.
- **Fail Action**: Block build, log design flaws.

## The Loop
1. Build.
2. `integrity-audit`.
3. `aesthetic-audit`.
4. Approved ONLY if both pass.
