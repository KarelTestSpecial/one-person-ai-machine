# Project Briefing: Nexus Trading Terminal

## Client Requirements (High Priority)
1. **Premium Aesthetic**: Must use glassmorphism, dark mode, and high-quality typography (Inter).
2. **Functional Sidebar**: Navigation must work; switching tabs should change the main view.
3. **No Dead Buttons**: Every action (Buy, Sell, Update) must show a visual response (Toast/Notification).
4. **Real-Time Simulation**: 
    - Must handle high-frequency data updates (10+ per second).
    - Must include a "Live" connection indicator with pulse animation.
    - Must display connection latency (ms) to prove performance stability.
5. **Interactive Charts**: Charts must not lag during high-frequency updates and should look professional.

## Evaluation Criteria
- **Pass**: All 5 requirements are visibly and functionally present.
- **Fail**: Any missing indicator, laggy UI, or non-functional button.
