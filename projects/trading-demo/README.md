# Apex AI: Stock Trading Dashboard

A high-performance, visually stunning stock trading intelligence dashboard built with React. This application features a modern "Glassmorphism" interface and real-time data simulation to provide an immersive market analytics experience.

## 🌟 Overview

Apex AI is a prototype trading dashboard designed for speed and clarity. It utilizes a premium shell architecture to deliver real-time market insights through a refined, dark-themed user interface. The project demonstrates advanced layout techniques, including CSS Backdrop Filters and responsive grid systems, to create a "glass" effect common in modern high-end fintech applications.

## 🛠 Technical Stack

- **Framework:** [React 18+](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS with Glassmorphism (Backdrop Filter) and Flexbox/Grid layouts.
- **Package Manager:** [pnpm](https://pnpm.io/)

## ✨ Key Features

- **Real-time Market Simulation:** Uses React hooks (`useEffect`, `useState`) to simulate live price fluctuations across various assets (Stocks, Crypto).
- **Glassmorphism UI:** A premium look and feel achieved through backdrop filters and semi-transparent layers.
- **Premium Dashboard Shell:** A reusable `PremiumShell` component providing sidebar, nav, and glass panels.
- **AI-Powered Analytics Engine:** A dedicated analytics area featuring animated bar charts.

## 🚀 Setup Instructions

Ensure you have [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/) installed.

1.  **Install dependencies:** `pnpm install`
2.  **Start dev server:** `pnpm dev`
3.  **Build:** `pnpm build`
