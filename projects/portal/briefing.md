# Project Briefing: One-Person AI Machine Portal

## Objective
Create a premium, high-converting gateway that showcases our autonomous development capabilities.

## Client Requirements
1. **Visual Excellence**: Dark theme, glassmorphism, and smooth Framer Motion transitions.
2. **Functional Navigation**: 
    - **Home**: Vision and value proposition.
    - **Machine**: Explanation of the 5-Gate Audit System.
    - **Portfolio**: Displaying the `trading-demo` and other successes.
    - **Contact**: Functional lead capture.
3. **Interactive Components**: 
    - A "Live Status" indicator showing the machine is operational.
    - Dynamic "Critical Audit" logs simulation to show the process in action.
4. **Performance**: Fast loading, zero lag, fully responsive.

## Evaluation Criteria
- **Pass**: All navigation tabs work, design is premium, and the "Machine" logic is visually explained.
- **Fail**: Placeholder text, broken links, or generic design.
