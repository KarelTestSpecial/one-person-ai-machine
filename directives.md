# DIRECTIVE: Build The One-Person Machine Portal

## Goal
Build a high-end, responsive landing page for 'The One-Person AI Machine' business. This is our primary portfolio piece to show clients what we can do in minutes.

## Requirements
- Use the `PremiumShell` master component.
- Content: Services (AI Automation, Web Development, Cloud Migration), Tech Stack (Gemini, Node, React), and a 'Get Started' CTA.
- Design: Dark mode, glassmorphism, accent colors (cyan/blue).
- Stack: Vite + React.

## Execution
1. Scaffold a new Vite project in `projects/portal`.
2. Import `PremiumShell.jsx` from the library.
3. Build the `App.jsx` with the required content.
4. Run a build check.
